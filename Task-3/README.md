# 🩺 MedBot: Medical Q&A Chatbot (Task 3)

**AI-powered medical question answering chatbot using the MedQuAD dataset**

---

## 📋 Overview

This is a specialized medical Q&A chatbot that retrieves relevant answers from the MedQuAD dataset using TF-IDF-based retrieval and performs basic medical entity recognition.

---

## 🎯 Task Requirements Fulfilled

| Requirement | Implementation |
|-------------|----------------|
| **Retrieval Mechanism** | TF-IDF scoring + keyword overlap to find relevant answers from 981 medical QA pairs |
| **Medical Entity Recognition** | Detects medical terms in 4 categories: Diseases, Symptoms, Treatments, Body Parts |
| **Streamlit UI** | Interactive chat interface with confidence scores, metrics, and entity highlighting |

---

## 🚀 Features

- ✅ **Smart Search**: TF-IDF-based retrieval finds the most relevant medical answers
- ✅ **Top 3 Results**: Shows multiple answer options with confidence scores
- ✅ **Medical Entity Detection**: Automatically identifies diseases, symptoms, treatments, and body parts
- ✅ **Confidence Metrics**: Displays TF-IDF score, keyword overlap, and confidence percentage
- ✅ **MedQuAD Dataset**: Uses 981 medical Q&A pairs from the official MedQuAD dataset
